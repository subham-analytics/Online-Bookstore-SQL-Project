# Online Bookstore SQL Project

## Project Overview

This project demonstrates the use of SQL and PostgreSQL to design and analyze an Online Bookstore database. The database consists of books, customers, and orders, allowing the extraction of valuable business insights related to sales, inventory, customer behavior, and revenue generation.

## Database Structure

The project contains three main tables:

* Books
* Customers
* Orders

## Tools Used

* PostgreSQL
* SQL
* Excel/CSV Datasets

## Key Analysis Performed

* Retrieve books by genre and publication year
* Analyze customer information by country and city
* Track orders and sales performance
* Calculate total revenue generated
* Identify the most frequently ordered books
* Find top customers based on spending
* Analyze genre-wise and author-wise book sales
* Monitor inventory and remaining stock levels

## SQL Concepts Used

* SELECT Statements
* WHERE Clauses
* Aggregate Functions (SUM, AVG, COUNT)
* GROUP BY and HAVING
* ORDER BY
* JOINS (INNER JOIN, LEFT JOIN)
* DISTINCT
* Date Filtering
* Inventory Analysis

## Key Insights

* Identified top-selling books and genres.
* Measured total revenue generated from customer orders.
* Analyzed customer purchasing behavior.
* Evaluated author performance based on books sold.
* Calculated remaining inventory after order fulfillment.
